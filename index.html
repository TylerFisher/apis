API

Application Programming Interface

It's time to go back to how the Internet works!

Client
A computer that makes requests
Server
A computer that responds to requests

REST

Representational State Transfer

There are different types of APIs, but we're going to work with RESTful APIs.

APIs are a way for your applications to talk to servers.

RESTful APIs work on top of HTTP, just like your web browser.

With APIs, we can send and receive data from servers for our own use.

Making API Requests

The Four Verbs of HTTP

GET
Retrieve information
POST
Send new information
PUT
Update existing information
DELETE
remove existing information

Every API request will utilize one of these verbs.

Let's make a GET Request

We'll use the NYTimes Congress API

bit.ly/nytcongress

Navigating API Documentation

Base URL
The URL that all of your requests will be based off of.
Response format
Know what format you want your data in. You probably want JSON.
Sample Request URLs
Look for sample request URLs and use those as starting points.
http://api.nytimes.com/svc/politics/{version}/us/legislative/congress/members/{chamber}/{state}/{district*}/current[.response-format]?api-key={your-API-key}

Accessing JSON

JavaScript Object Notation

Objects are key-value stores. You access an object's value by referencing its key

{
    "key": "value",
    "key_to_object": {
        "yet_another_key": "yet_another_value",
        "key_key_key": "value_value_value"
    },
    "key_to_array": [
        {
            "key": "value",
            "another_key": "another_value"
        },
        {
            "key": "value",
            "another_key": "another_value"
        }
    ]
}

And in JavaScript...

$.getJSON('apiURL', function(data) {
    // store your results!
    var results = data.results;

    // loop through your results!
    $.each(results, function(key, value) {
        console.log(key, value);
    });
});

What does this return?

Access particular values in your results

$.getJSON('apiURL', function(data) {
    // store your results!
    var results = data.results;

    // loop through your results!
    $.each(results, function(key, value) {
        console.log(value.twitter_id);

        // or!
        console.log(value['twitter_id']);
    });
});
